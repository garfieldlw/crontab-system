#Crontab System


## 简介
任务管理系统

## 代码说明
* web：后台管理页面
* app：后台管理api
* cron/master：控制管理节点
* cron/worker: 工作节点

## 功能简单介绍
支持以下主要功能
* 串行任务
* 并行任务
* 任务依赖
* 任务重试
* 多工作节点
* 工作节点任务可配置
* 定时任务
* 实时任务


## 流程
1. 定时/实时任务触发工作流
2. master根据任务流配置发布任务
3. worker节点竞争任务，并把任务结果反馈给master
4. master处理任务结果，如果还有下级任务，执行2，否则执行5
5. 处理结束，推送通知
